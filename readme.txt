== Description ==

"WP Testimonial" is a very simple plugin for add to "Testimonials" in sidebar widget OR any page/post (using shortcode).

= Features =

 * Rotator Widget
 * Shortcode for "Testimonial Rotator" & "Testimonial List"
 * Option for manage the Effect

= Shortcode =

 * [wt_testimonials] : Display all Testimonials
 * [wt_random] : Add rotator on any post/page


== Installation ==

Step 1. Upload wp-testimonial folder to the `/wp-content/plugins/` directory

Step 2. Activate the plugin through the Plugins menu in WordPress

== Frequently Asked Questions ==

* How Configure the the WP Testimonial settings?

Go to "Settings/WP Testimonial" and configure the plugin settings

* How Add/Edit the testimonial posts from admin?

Find the "WP Testimonial" (given in under the right side menu) and add/edit your new testimonial posts

* How add the WP testimonial rotator widget in my site sidebar?

Find the "WP Testimonial" Widget from "Appearance >> Widgtes >> WP Testimonial" and add widget where you want to show it.

* Can i show all testimonials on any specific page?

Yes,you have just need to create a new testimonial page and add this [wt_testimonials] shorcode, after add this shortcode all testimonials will be publish on that page

* Can i add the rotator on any page/post not in widget?

Yes,you can add the rotator on any post/page by using [wt_random] shortcode
